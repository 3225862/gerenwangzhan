# 内容维护手册

本地运行 `npm run dev`，打开 `/admin`。首页在“首页内容”，个人资料在“个人资料”，网站导航在“网站设置”。项目和文章分别在对应集合中编辑。草稿保存后不会出现在公开构建；确认标题、摘要、slug、图片格式和公开开关后再取消草稿。

图片只使用 jpg、jpeg、png、webp，单张不超过 2MB，放在对应公开图片目录。出错时先阅读构建检查提示；回滚使用 Git 恢复上一份已确认内容。发布前备份整个仓库。

视频只填写外部链接，不上传大视频。小型 PDF、PPT、Word 可以放在 `public/downloads`，大文件使用外部存储。图片直接在对应内容的图片字段选择，不需要重复上传同一张图片。日常维护不需要修改代码或 YAML。
