# Website Launcher

## 启动

双击 `scripts\launch.bat`，或双击桌面的 `个人网站.lnk`。启动器会自动定位项目目录、检查 Node/npm、按需安装依赖，并持续检测 `http://localhost:4321`，确认可访问后打开 `/admin`。

如果网站已经运行，启动器不会重复启动第二个 Astro，只会打开后台。启动窗口保持打开，关闭该窗口即可结束本次网站启动会话。

## 停止

双击 `scripts\stop.bat`。它只会停止启动器记录且命令行属于当前项目目录的进程，不会按名称误杀其他 Node 进程。

## 状态

运行 `scripts\status.ps1` 可查看网站状态、Node/npm/Astro 版本、项目目录、4321 端口和后台地址。

## 常见问题

- 找不到 Node 或 npm：安装 Node.js LTS 后重新启动。
- 依赖安装失败：检查网络和项目目录写入权限后重试。
- 4321 被其他程序占用：启动器不会接管其他程序，请先释放端口。
- 首次启动较慢：启动器会持续检测服务响应，不使用固定等待时间。

项目移动到其他盘符或用户目录后，重新运行 `scripts\launch.bat` 即可重新生成桌面快捷方式。
