# Windows 本地运行指南

## 第一步：解压

把 ZIP 解压到一个容易找到的文件夹，例如 `C:\\website`。

## 第二步：安装 Node.js

安装 Node.js LTS。打开 PowerShell，输入 `node -v`，能看到版本号就可以继续。

## 第三步：安装依赖

在项目文件夹中打开终端，输入：

```powershell
npm.cmd ci
```

## 第四步：启动网站

```powershell
npm.cmd run dev
```

浏览器打开 `http://127.0.0.1:4321`。后台入口是 `/admin`，编辑页面是 `/keystatic`。

## 常见问题

- PowerShell 不允许运行 `npm.ps1` 时，使用 `npm.cmd`。
- 4321 被占用时，终端会提示新的端口，按提示打开。
- 修改内容后刷新浏览器；开发服务器会自动重新读取内容。
- 关闭终端后网站会停止，这是正常现象；再次运行启动命令即可。
- 页面打不开时，先确认终端仍在运行，并确认地址是 `127.0.0.1:4321`。
